查找一个字符或一段指定的文本。若找到匹配，则该函数会返回第一个匹配的字符位置。若没有找到匹配，则返回FALSE。

PHP比较运算符：

== 等于

=== 绝对等于

!= 不等于

<>不等于

!==绝对不等于

PHP逻辑运算符

and 与

or 或

xor 异或

&& 与

|| 或

! 非

PHP数组运算符

x+y (集合) x和y的集合

x==y（相等）如果x和y具有相同的键/值对，则返回true

x===y（恒等）如果x和y具有相同的键/值对，且顺序相同类型相同，则返回true

三元运算符

**”?:”**条件运算符

对 expr1 求值为 TRUE 时的值为 expr2，在 expr1 求值为 FALSE 时的值为 expr3。

组合比较符（太空船运算符）：

<=>

若$a>$b,则$c的值为1

$a==$b，则$c的值为0

$a<$b，则$c的值为-1
